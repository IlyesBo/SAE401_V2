Politique de Confidentialité

Conformité RGPD - Sushill.fr
Chez Sushill.fr, nous prenons la protection de vos données personnelles très au sérieux. Cette politique de confidentialité explique quelles données nous collectons, pourquoi nous les collectons, comment nous les utilisons, les mesures que nous prenons pour protéger ces données, et vos droits en tant qu'utilisateur de notre site web. Données personnelles collectées Sur Sushill.fr, nous collectons les types de données personnelles suivants :
● Adresse e-mail : Pour vous permettre de créer un compte, passer des commandes et recevoir des confirmations de commande.
● Mot de passe : Pour sécuriser votre compte sur notre site web.
● Historique de commande : Pour améliorer notre service client et vous offrir une expérience utilisateur personnalisée.
● Articles ajoutés au panier : Pour faciliter le processus de commande et vous permettre de retrouver vos articles sélectionnés lors de vos visites ultérieures. Utilisation des données personnelles
Les données personnelles collectées sur Sushill.fr sont utilisées dans le but suivant : ● Améliorer votre expérience utilisateur sur notre site web.
● Traiter et exécuter vos commandes.
● Vous informer sur le statut de vos commandes.
● Répondre à vos demandes de service client.
● Analyser et améliorer nos services.
Stockage et protection des données Vos données personnelles sont stockées de manière sécurisée localement votre appareil. Nous prenons des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, utilisation abusive ou divulgation.
Partage des données personnelles
Nous ne partageons pas vos données personnelles avec des tiers, sauf dans les cas suivants :
● Avec nos prestataires de services de paiement pour traiter vos transactions.
● Lorsque requis par la loi ou en réponse à une demande légale.
Conservation des données
Nous conservons vos données personnelles aussi longtemps que nécessaire pour fournir les services demandés et respecter nos obligations légales. Vous avez le droit de demander la suppression de vos données personnelles à tout moment. Vos droits En tant qu'utilisateur de Sushill.fr, vous avez les droits suivants en ce qui concerne vos données personnelles :
● Droit d'accès : Vous pouvez demander à accéder aux données personnelles que nous détenons à votre sujet.
● Droit de rectification : Vous pouvez demander la correction de toute information inexacte ou incomplète.
● Droit à l'effacement : Vous pouvez demander la suppression de vos données personnelles dans certaines circonstances.
● Droit d'opposition : Vous pouvez vous opposer au traitement de vos données personnelles dans certaines situations. ● Droit à la portabilité des données : Vous pouvez demander la transmission de vos données personnelles à un autre contrôleur de données dans un format structuré et couramment utilisé.
